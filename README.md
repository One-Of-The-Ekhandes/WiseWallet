# 💸 Wise Wallet App

A simple and elegant web-based expense tracker built with Python, Flask, and CSV.

## 🚀 Features
- Add your daily expenses
- View all past transactions
- Categorized spending summary

## 🛠️ Tech Stack
- Python 🐍
- Flask 🔥
- HTML + Bootstrap 🎨
- CSV for data storage
